HTML

  • In VS Code, you can use "!" to quickly get the structure of an html doc, but its bad practice in the beginning.
  • The head element contains information about the webpage.
  • In the head section, we define character set, viewport, title, and many more meta tags.
  • The body element represents the visible content shown to the user.
  • In the body section, we use h1-h6 to indicate headers from greater size to less.
  • Further, we can use p for paragraph, lists (unorderd lists, ordered lists).
  • img is also a very important html element to display an image along with using src for the source of the image.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A border is between the margin and padding surrounding the content.
  • External CSS is the most common way to style a webpage.
  • CSS Rule: selector { property : value; }
  • multiple selectors can be chosen with commas.
  • To link external styling sheet: use link element and rel="stylesheet" href="./locationOfFile"
